## 一、概念

> 核心概念与定义

### 概念 1：前馈控制（Feedforward Control）

- **定义：** 基于扰动来消除扰动对被控量的影响，又称为"扰动补偿"。当扰动发生后，前馈控制器根据扰动的性质和大小，通过控制通道施加相反的作用，抵消扰动对被控变量的影响。
- **本质：** 按扰动调节的开环控制
- **核心理念：** 预防——在扰动影响被控量之前就进行补偿
- **对偶概念：** [[反馈控制]]（按偏差调节的闭环控制）

### 概念 2：不变性原理

- **定义：** 控制系统的被调量与扰动量无关，或在一定准确度下无关。
- **数学表达：** 当 $D_i(t) \neq 0$ 时，$y(t) = C$（恒定）
- **四种不变性：**

| 类型 | 定义 | 工程应用 |
|------|------|----------|
| **绝对不变性** | 整个过程中动态偏差均为零 | 工程上无法实现 |
| **误差不变性（ε不变性）** | 偏差限制在一定误差 ε 内 | 反馈控制即为 ε 不变性 |
| **稳态不变性** | 稳态下偏差为零，动态偏差不为零 | 控制要求不高的场合 |
| **选择不变性** | 对其中几个主要干扰实现不变性 | 节省投资，实用 |

### 概念 3：前馈控制器模型（不变性条件）

- **推导：** 希望扰动 $D(s) \neq 0$ 时 $Y(s) = 0$

$$Y(s) = G_D(s) \cdot D(s) + G_{ff}(s) \cdot G_v(s) \cdot G_p(s) \cdot D(s) = 0$$

- **前馈控制器传递函数：**

$$G_{ff}(s) = -\frac{G_D(s)}{G_p(s) \cdot G_v(s) \cdot G_t(s)}$$

- **简化形式（设 $G_v = G_t = 1$）：**

$$G_{ff}(s) = -\frac{G_D(s)}{G_p(s)}$$

### 概念 4：静态前馈 vs 动态前馈

| | 静态前馈 | 动态前馈 |
|------|----------|----------|
| **控制器形式** | 比例控制器 $K_{ff}$ | 超前-滞后环节 |
| **公式** | $K_{ff} = -\frac{K_D}{K_p}$ | $G_{ff}(s) = K_{ff}\frac{T_p s + 1}{T_D s + 1}$ |
| **补偿效果** | 只消除静态偏差 | 消除动态+静态偏差 |
| **实现难度** | 简单，可用常规仪表 | 复杂，需专用控制器 |
| **适用场景** | $0.7 < T_D/T_p < 1.3$ | 干扰频繁且动态精度要求高 |

### 概念 5：前馈控制器的超前-滞后特性

- **$T_p > T_D$**：前馈控制器呈**超前**特性
- **$T_p < T_D$**：前馈控制器呈**滞后**特性
- **$T_p = T_D$**：前馈控制器呈**比例**特性

## 二、前馈与反馈的对比

> 前馈控制与反馈控制的本质区别

| 比较项目 | 反馈控制 | 前馈控制 |
|----------|----------|----------|
| **设计原理** | 反馈控制理论 | 不变性原理 |
| **被测变量** | 被控变量（CV） | 扰动量（D） |
| **控制依据** | 设定值与测量值的偏差 | 被测扰动的大小 |
| **系统结构** | 闭环 | 开环 |
| **典型控制器** | P、PI、PD、PID | 超前-滞后环节、比例 |
| **作用时机** | 扰动产生影响**之后** | 扰动产生影响**之前** |
| **稳定性** | 需要分析，可能不稳定 | 各环节稳定则系统必然稳定 |
| **可克服的扰动** | 回路内**所有**扰动 | 只能克服**单一**可测扰动 |
| **控制规律** | 通用 PID，与对象无关 | 取决于被控对象特性（专用） |
| **能否单独使用** | ✅ 可以 | ❌ 通常不单独使用 |

### 反馈控制的优缺点

- ✅ 对所有来源和类型的扰动有效
- ✅ 可消除稳态余差
- ✅ 对过程信息要求少
- ✅ 具有通用性和鲁棒性
- ❌ 只有被控变量偏离设定值后才能动作（不及时）
- ❌ 理论上不能达到"完美"控制
- ❌ 影响闭环稳定性

### 前馈控制的优缺点

- ✅ 在 CV 受影响**前**补偿扰动
- ✅ 理论上可达"完美"控制
- ✅ 不影响系统稳定性
- ❌ 扰动必须可测（增加成本）
- ❌ 对过程模型要求高
- ❌ 理想控制器物理上可能不可实现
- ❌ 只能克服单一扰动

### 前馈控制的局限性

1. 只能克服**可测不可控**的干扰
2. 实际过程中干扰众多，为每个干扰设前馈不现实
3. 前馈控制规律取决于过程特性，精确获取困难
4. 过程非线性导致原前馈控制器失效

## 三、前馈控制系统结构形式

### 结构 1：单纯前馈控制系统

- 开环控制系统
- **不常单独采用**
- 与开环比值控制系统相似

### 结构 2：前馈-反馈复合控制系统（最常用）

- **前馈控制：** 消除可测扰动的影响（粗调）
- **反馈控制：** 校正不可测扰动、模型误差等（微调）

**传递函数：**
$$Y(s) = \frac{G_c G_0}{1 + G_c G_0}R(s) + \frac{G_D + G_{ff} G_t G_v G_p}{1 + G_c G_0}D(s)$$

**关键结论：** 前馈控制器的加入**不影响**反馈回路的稳定性（特征方程中不含 $G_{ff}$）

### 结构 3：前馈-串级复合控制系统

- 结合串级控制和前馈控制的优势
- 适用于多个变化频繁且剧烈的干扰，同时控制精度和稳定性要求高
- **前馈控制器：** $G_{ff}(s) \approx -\frac{G_D(s)}{G_{01}(s)}$（设 $G_{02}^* \approx 1$）

### 结构 4：前馈与反馈相乘（比值控制型）

- 实质是比值控制系统
- 一个流量（扰动）去改变另一个流量（操纵变量）

## 四、方法——前馈控制系统设计

> 设计规则、步骤与技巧

### 4.1 五个设计规则

当存在以下情况时可采用前馈控制：

| 序号 | 条件 | 说明 |
|------|------|------|
| **规则 1** | 单回路控制系统性能**不能满足要求** | 反馈控制效果差 |
| **规则 2** | 有一个被测变量是**有效的**（扰动可测） | 在线测量 |
| **规则 3** | 能够代表**重要扰动**的发生 | 主要干扰源 |
| **规则 4** | 扰动变量与执行机构（阀门）**没有因果关系** | 不可控性 |
| **规则 5** | 被控变量 CV **不能**比操作变量 MV 有更快的响应 | 补偿来得及 |

**记忆口诀：** 反馈不够好 → 扰动可测 → 是主要扰动 → 不能通过阀门控制 → 响应不能太快

### 4.2 扰动变量的选择原则

- 扰动必须**可测**（有在线测量手段）
- 扰动应是**不可控**或不宜控制的
- 扰动应是**主要扰动**（变化频繁、幅度大）
- 主要干扰无法纳入串级副环时，优先选前馈
- 为克服其他扰动，通常采用**前馈-反馈复合**系统

### 4.3 前馈控制器设计方法

#### （1）静态前馈控制器设计

**热平衡法（以换热器为例）：**
$$W_H C_H(\theta_0 - \theta_i) = W_s \Delta H$$

**静态前馈增益：**
$$K_{ff} = -\frac{K_D}{K_p}$$

#### （2）动态前馈控制器设计

**一阶纯滞后对象：**
$$G_{ff}(s) = K_{ff} \cdot \frac{T_p s + 1}{T_D s + 1} \cdot e^{-\theta_{ff} s}$$

其中：
- 增益：$K_{ff} = -\dfrac{K_D}{K_p}$
- 超前时间：$T_1 = T_p$
- 滞后时间：$T_2 = T_D$
- 纯滞后：$\theta_{ff} = \theta_p - \theta_D$（$\theta_p \geq \theta_D$）

### 4.4 选用原则

1. **必要条件：** 扰动量可测且不可控
2. 过程控制通道时间常数 > 干扰通道时间常数，且反馈不及时时
3. 主要干扰无法用串级包含在副回路内时
4. **经济性原则：** 静态前馈能满足要求则不使用动态前馈

## 五、方法——参数整定

> 前馈控制系统的调试与参数整定

### 5.1 静态前馈增益 $K_{ff}$ 的整定

**重要性：** $K_{ff}$ 过小 → 欠补偿；$K_{ff}$ 过大 → 过补偿，相当于给反馈回路施加干扰

#### 开环整定法

1. 断开反馈回路，系统处于单纯静态前馈状态
2. 施加阶跃干扰
3. $K_{ff}$ 由小逐步增大
4. 直到被控变量回到给定值 → 此时的 $K_{ff}$ 为最佳值

#### 闭环整定法（方法 1：试凑法）

1. 开关 K 打开 → 只有反馈控制 → 整定反馈控制器参数
2. 开关 K 闭合 → 前馈-反馈状态 → 施加同样干扰
3. 由小到大改变 $K_{ff}$，观察输出曲线：
   - **欠补偿：** 增大 $K_{ff}$
   - **过补偿：** 减小 $K_{ff}$
4. 直到补偿合适

#### 闭环整定法（方法 2：差值法）

1. 只有反馈控制时，施加扰动 $N_0$，记录 $u_0$
2. 改变扰动为 $N_1$，记录 $u_1$
3. 计算：$K_{ff} = -\dfrac{u_1 - u_0}{N_1 - N_0}$
4. 设置 $K_{ff}$，施加扰动，微调

### 5.2 动态时间常数 $T_1$、$T_2$ 的整定

$$G_{ff}(s) = K_{ff} \cdot \frac{T_1 s + 1}{T_2 s + 1}$$

| 曲线特征 | 原因 | 调整方法 |
|----------|------|----------|
| **欠补偿**（第一峰值未明显减小） | $T_1$ 过小 / $T_2$ 过大 | 增大 $T_1$ 或 减小 $T_2$ |
| **过补偿**（出现反向过调） | $T_1$ 过大 / $T_2$ 过小 | 减小 $T_1$ 或 增大 $T_2$ |
| **完全补偿** | $T_1 = T_p$，$T_2 = T_D$ | 保持 |

⚠️ **重要原则：**
- **过补偿是前馈控制系统危险之源**，可能破坏控制过程
- 整定应**从欠补偿开始**，逐步增强前馈作用
- 直至出现过补偿趋势，再略减前馈作用，获得满意过程

### 5.3 偏置值设置

$$B = -K_{ff} \cdot K_m \cdot \bar{D}$$

偏置值 = 正常工况下扰动经前馈控制器后的输出的**负值**，用于抵消正常工况下的输出。

## 六、例题

> 典型应用与分析

### 例题 1：换热器前馈-反馈控制系统设计 ^ex-1

**【题目】**

图 5-6 为一单回路水位控制系统。如果蒸气用量经常发生变化，为了改善控制质量，将单回路控制系统改为前馈-反馈复合控制系统。

**【思路分析】**

1. 蒸气用量是主要扰动 → 可测（流量计）且不可控（由下游工艺决定）
2. 满足前馈设计五个规则
3. 前馈信号（蒸气流量）与反馈控制器输出相加，控制给水阀

**【解答】**

**工艺流程图（前馈-反馈复合）：**

- 前馈回路：蒸气流量 FT → 前馈控制器 Gff → 与反馈输出相加 → 控制给水阀
- 反馈回路：水位 LT → 反馈控制器 Gc → 与前馈输出相加 → 控制给水阀

**新增回路功能：**
- 蒸气流量变化时，不等水位偏差出现，前馈控制器直接调整给水量
- 反馈回路负责消除其他扰动及模型误差带来的水位偏差

**【关键点】**

- 前馈克服主要扰动（蒸气量变化），反馈消除余差和其他扰动
- 前馈不影响系统稳定性
- 是前馈-反馈**相加**型结构

---

### 例题 2：静态前馈增益计算 ^ex-2

**【题目】**

换热器中，蒸汽汽化潜热 $\Delta H$，物料比热 $C_H$，热平衡方程：
$$W_H C_H(\theta_0 - \theta_i) = W_s \Delta H$$

求静态前馈增益 $K_{ff}$。

**【解答】**

从静态考虑：
$$\theta_0 = \frac{W_s \Delta H}{W_H C_H} + \theta_i$$

**控制通道静态增益：**
$$K_p = \frac{\partial \theta_0}{\partial W_s} = \frac{\Delta H}{W_H C_H}$$

**扰动通道静态增益：**
$$K_D = \frac{\partial \theta_0}{\partial W_H} = -\frac{W_s \Delta H}{W_H^2 C_H} = -\frac{\theta_0 - \theta_i}{W_H}$$

**前馈增益：**
$$K_{ff} = -\frac{K_D}{K_p} = \frac{W_s}{W_H} = \frac{C_H(\theta_0 - \theta_i)}{\Delta H}$$

## 七、工程应用注意事项

1. **前馈信号线性化：** 流量信号应加开方器，温度信号需非线性校正
2. **动态前馈加输出限幅：** 从安全角度出发
3. **一般优先采用静态前馈+反馈控制**
4. **动态前馈参数整定不匹配时效果更差**
5. **投运顺序：** 先投反馈控制，再投前馈控制
6. **DCS 实现：** Honeywell PIDFF、Foxboro LLAG+DTIME、ABB REG 等

## 四、反思

### ⚠️ 易混淆概念

| 易混淆点 | 正确理解 |
|----------|----------|
| 前馈是闭环？ | ❌ 前馈是**开环**控制 |
| 前馈影响稳定性？ | ❌ 前馈**不影响**反馈回路稳定性 |
| 前馈只能用于一种扰动？ | ✅ 一套前馈只能克服**一种**扰动 |
| 前馈能完全替代反馈？ | ❌ 前馈通常**不能单独使用** |

### 核心理解

1. **不变性原理**是前馈控制的理论基石
2. 前馈 = **预防**（扰动来之前行动），反馈 = **治疗**（偏差出来再行动）
3. 前馈+反馈 = 最佳组合（前馈粗调 + 反馈微调）
4. 五个设计规则是判断能否用前馈的**核心工具**
